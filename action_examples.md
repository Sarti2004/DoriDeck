# DoriDeck usage examples

Some ideas for DoriDeck buttons.

## Duplicate Current Flow

Duplicate the currently selected flow.

## Remove Dynamics in All Flows

Remove all dynamics from every flow in the project.

## Run a Script or Action in All Flows

Run the same script, macro, or Dorico command in every flow.

## Move Bar to Next System

DoriDeck → Run Command → `Page.MoveBarsToNextSystem`

## Move Bar to Previous System

DoriDeck → Run Command → `Page.MoveBarsToPreviousSystem`

## Remove Rests

DoriDeck → Run Command → `Edit.RemoveRests`

## Keep Top Notes Only

Reduce selected chords to their highest notes.

Useful for quickly creating a melody line from chordal material.

## Keep Bottom Notes Only

Reduce selected chords to their lowest notes.

Useful for extracting a bass line.

## Remove Articulations

Remove articulations from the current selection or from all flows.

## Remove Playing Techniques

Clear playing techniques from the selected music.

## Remove Slurs

Remove all slurs from the current selection.

## Make Notes Voice 1

Move selected notes into the first voice.

## Swap Voices

Swap the selected music between voices.

## Hide Stems

Hide stems for selected notes.

## Show Stems

Restore stems for selected notes.

## Force Stems Up

Set the stems of selected notes to point upward.

## Force Stems Down

Set the stems of selected notes to point downward.

## Add Cue

Create a cue from another instrument using a predefined sequence of commands.

## Add System Break

Insert a system break at the selected rhythmic position.

## Move Selected Bars to Next System

DoriDeck → Run Command → `Page.MoveBarsToNextSystem`

## Move Selected Bars to Previous System

DoriDeck → Run Command → `Page.MoveBarsToPreviousSystem`

## Toggle Signposts

Show or hide signposts.

## Toggle Invisibles

Show or hide hidden items and other invisible notation elements.

## Open Engraving Options

Open Engraving Options.

## Open Notation Options

Open Notation Options for the current flow.

## Open Layout Options

Open Layout Options for the current layout.

## Transpose Up an Octave

Transpose the selected music up one octave.

## Transpose Down an Octave

Transpose the selected music down one octave.

## Transpose by a Preset Interval

Create separate buttons for commonly used intervals, such as:

* Up a major second
* Down a major second
* Up a perfect fourth
* Down a perfect fifth

## Add a Specific Dynamic

Create dedicated buttons for frequently used dynamics:

* `pp`
* `p`
* `mp`
* `mf`
* `f`
* `ff`

## Add Common Playing Techniques

Create buttons for frequently used techniques, such as:

* Pizzicato
* Arco
* Mute
* Open
* Sul ponticello
* Sul tasto

## Add Common Ornaments

Create buttons for:

* Trill
* Mordent
* Turn
* Tremolo

## Add Common Text

Insert frequently used staff or system text, such as:

* Solo
* Tutti
* Div.
* Unis.
* Simile
* Optional
* Cue

## Open the Jump Bar with a Preset Query

Open the Jump Bar and enter a predefined command or search term.
